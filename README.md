# FlowtileWM

FlowtileWM это `docs-first` проект оконного менеджера и shell-подсистемы для `Windows 11`, вдохновлённый `niri` и стремящийся воспроизвести максимально возможную часть его пользовательской модели в рамках ограничений Windows.

Работа с окнами строится поверх `DWM` и не заменяет его.

Проект находится в активной разработке. Репозиторий уже содержит рабочую кодовую базу, но стабильные пользовательские сборки пока не публикуются, поэтому запуск сейчас предполагает сборку из исходников и не гарантирует полностью безошибочную работу на любой машине.

На текущем этапе реализована только часть запланированного функционала, и проект всё ещё заметно уступает `niri` по полноте возможностей.

Если вы хотите попробовать проект в текущем состоянии, собирайте его из исходников. Если при запуске или проверке поведения обнаружится дефект, его можно оформить через issue в этом репозитории.
