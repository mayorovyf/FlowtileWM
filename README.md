# FlowtileWM

FlowtileWM это docs-first проект оконного менеджера и shell-подсистемы для Windows с поведением, максимально близким к `niri`, но работающим поверх `DWM`, а не вместо него.

Сейчас репозиторий уже содержит стартовый кодовый каркас на `Rust` и `C# / WinUI 3`, а канонический источник истины для архитектуры, требований и release-линии находится в `docs/dev`, `docs/meta` и `docs/versions`.
